# AccessID – Disability Certificate Wallet (Portfolio Project)

AccessID is a React Native mobile application designed to securely store and
present disability certificates with an accessibility-first approach.

## Key Features
- Offline PDF certificate storage and viewing
- Biometric app lock
- QR-based demo verification
- Accessibility settings (large text, high contrast)
- Clean, scalable architecture (TypeScript)

## Tech Stack
React Native (Expo), TypeScript, React Navigation, Local Authentication

⚠️ This is a portfolio/demo project and does not represent an official
government verification system.
